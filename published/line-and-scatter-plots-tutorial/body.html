LINE AND SCATTER PLOT: MATLAB SYNTAX

Line and scatter plots are an excellent way to visualize underlying trend within data consisting of two variables. Typically, line and scatter plots are used to observe relationships between a measured parameter (dependent variable), whose values may vary as a function of an incremented control parameter (indepent variable). Line and scatter plots can also be used to visualize the strength and direction of the correlation between two variables, regardless of whether or not a clear dependent variable exists. For this line and scatter plot walkthrough, we will be considering the 2014 U.S. TRAVEL AND TOURISM STATISTICS on the monthly import / export of travel and tourism within the U.S.A. We begin by launching our MATLAB environment and importing THE COLLECTED DATA .

As we can see, the returned fields of the travelData structure array consist of imports.travel / imports.fare and exports.travel / exports.fare . Lets take a step back and ask: what is this data describing? For the purposes of this example, imports can be equated to all the money associated with travel-related good / services that the U.S.A spends (loses) and exports can be equated to all the money associated with any travel-related goods / services that the U.S.A. earns (gains). According to the descriptions included in the spreadsheet of THE COLLECTED DATA , we can break down each term as follows:

IMPORTS.TRAVEL : These accounts cover purchases of goods and services by U.S. persons traveling abroad.

IMPORTS.FARE : These accounts cover the fares paid by U.S. residents to foreign air carriers for travel between the United States and foreign countries and foreign vessel operators for travel on cruise vessels.

EXPORTS.TRAVEL : These accounts cover purchases by foreign travelers in the United States for business or personal reasons.

EXPORT.FARE : These accounts cover the fares received by U.S. air carriers from foreign residents for travel between the United States and foreign countries and between two foreign points, the fares received by U.S vessel operators for travel on cruise vessels.

BASIC SCATTER PLOT

Let's have a look at how to create our first Plotly scatter plot using the MATLAB API fig2plotly.m function. For more information regading fig2plotly.m and all other Plotly MATLAB API functions, consult the complete FUNCTION DOCUMENTATION . Take a moment to review the syntax if you are unfamilliar with this function call. We will begin by creating a scatter plot in MATLAB using the inherent scatter plot function:

>> scatter.m

BASIC SCATTER + LINE PLOT

Trend lines, or lines of best fit, are standard practice when it comes to describing a suspected linear relationship between two variables. Upon visual inspection of the above plot, there is a seemingly strong linear realtionship between the travel and fares expenditures .

In [96]: 
%%matlab
plot_help = help('plot'); 
plot_help_out = plot_help(1:320)


plot_help_out =

 PLOT   Linear plot. 
    PLOT(X,Y) plots vector Y versus vector X. If X or Y is a matrix,
    then the vector is plotted versus the rows or columns of the matrix,
    whichever line up.  If X is a scalar and Y is a vector, disconnected
    line objects are created and plotted as discrete points vertically at
    X.
In [98]: 
%%matlab
%---- FROM BEFORE ---- %

%new matlab figure
fig = figure; 

%first scatter plot colour
col1 = [50 200 200]/255; 

%scatter plot imports.travel vs. imports.fare
scatter(imports.travel,imports.fare,100,col1)

%on the same plot... 
hold on

%second scatter plot colour 
col2 = [100 20 250]/255; 

%scatter plot exorts.travel vs. exports.fare 
scatter(exports.travel,exports.fares,100, col2)

%Title/labels 
title('2014 U.S. Travel and Tourism Statistics')
xlabel('Travel (Millions USD)')
ylabel('Fares (Millions USD)')

%---- LINE PLOTS ---%

%line plot for imports: line of best fit
coeffs1 = polyfit(imports.travel,imports.fare, 1);

% Get fitted values
fittedX1 = linspace(min(imports.travel), max(imports.travel), 200);
fittedY1 = polyval(coeffs1, fittedX1);

%first trend line  colour 
col3 = [50 100 100]/255; 

%Plot the fitted line
hold on;
plot(fittedX1, fittedY1, 'LineWidth', 4,'Color',col3,'LineStyle','--');

%line plot for exports: line of best fit
coeffs2 = polyfit(exports.travel,exports.fares, 1);

% Get fitted values
fittedX2 = linspace(min(exports.travel), max(exports.travel), 200);
fittedY2 = polyval(coeffs2, fittedX2);

%second trend line colour 
col4 = [100 20 150]/255; 

% Plot the fitted line
hold on;
plot(fittedX2, fittedY2, 'LineWidth', 4,'Color',col4,'LineStyle','--');

%turn on the axes grid
grid

%add a legend with the newly created trend lines
legend('Imports','Exports','LBF: Imports','LBF: Exports','Location','BestOutside')

%turn off the legend border
legend boxoff

%       PLOTLY       %
%%%%%%%%%%%%%%%%%%%%%%

plot_help_out =

 TEXT   Text annotation.
    TEXT(X,Y,'string') adds the text in the quotes to location (X,Y)
    on the current axes, where (X,Y) is in units from the current
    plot. If X and Y are vectors, TEXT writes the text at all locations
    given. If 'string' is an array the same number of rows as the
    length of X and Y, TEXT marks each point with the corresponding row
    of the 'string' array.
